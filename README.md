# CPT Production Scheduler

Weighted heuristic volunteer scheduling engine built with:

- Python
- Streamlit
- Pandas
- XLSX / CSV / ODS support

## Features

- Fairness-first scheduling
- Multi-campus balancing
- Skill-based optimization
- ODS spreadsheet support
- Streamlit UI
- Excel exports
- Summary analytics
- GitHub-ready structure

## Deployment

### Local

```bash
pip install -r requirements.txt
streamlit run app.py
